# Re:Prompt

macOS 放置時間削減・行動喚起アプリ。

キーボード・マウスの無操作時間が閾値を超えると、画面中央にフローティングウィンドウを表示して行動再開を促します。何か操作すれば即座に閉じます。

## 動作環境

- macOS 13 (Ventura) 以降
- Swift 5.8+

## ビルド・実行

```bash
# ビルド（コマンドライン実行用）
make build

# ビルド＋実行
make run

# .app バンドル作成（Finder からダブルクリックで起動可能）
make app

# /Applications にインストール
make install

# クリーン
make clean
```

## 使い方

1. 起動するとメニューバーに「Re:P」が表示される（Dock には出ない）
2. 無操作時間が閾値を超えるとウィンドウが表示される
3. キーボードかマウスを操作すると閉じる
4. 再び放置すると再表示される

### メニュー

| 項目 | 内容 |
|---|---|
| 閾値 | 30秒〜30分から選択（変更は自動保存される） |
| メディアフォルダを開く | メディア格納先を Finder で開く |
| 終了 | アプリを終了 |

## メディア

`~/Library/Application Support/RePrompt/Media/` にファイルを配置すると、ウィンドウ表示時に再生されます。

| 種類 | 対応形式 | 動作 |
|---|---|---|
| 動画 | `.mp4` `.mov` `.m4v` | ループ再生（優先） |
| 画像＋音声 | `.jpg` `.png` + `.mp3` `.m4a` `.wav` | 画像表示＋音声ループ再生 |
| なし | — | ダーク画面＋システム音 |

画像・動画の上には「何やってるんですか仕事してください」のテキストがオーバーレイされます。

## ファイル構成

```
Sources/RePrompt/
  main.swift                  # エントリポイント
  AppDelegate.swift           # ステータスバー、ライフサイクル管理
  IdleMonitor.swift           # IOKit HIDIdleTime によるアイドル検知
  AlertWindowController.swift # ウィンドウ表示・メディア再生
  Config.swift                # 設定値（閾値、ウィンドウサイズ等）
```
